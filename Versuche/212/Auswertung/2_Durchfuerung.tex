\section{Messverfahren}
\label{sec:Messverfahren}

\paragraph{Stokes}
Im ersten Versuchsteil werden POM-Kugeln verschiedener Durchmesser möglichst mittig in den Fallzylinder fallengelassen, welcher mit PEG Gefüllt ist. Es wird sich für jede Kugel eine Fallstrecke heruasgesucht, welche an der Skala abgelesen werden kann. 
Für diese Strecke wird die Sinkzeit der jeweiligen Kugel per Stoppuhr gemessen. Größere Kugeln sinken scheller, daher wird hier eine größere Strecke gewählt, um die Messfehler kleinzuhalten. Jeder Kugeldurchmesser wird fünf mal gemessen und final wird die durchschnittliche Sinkgeschwindigkeit bestimmt. 
Es gibt neun Durchmesser, von $1\mathrm{mm}$ bis $9\mathrm{mm}$. Die jeweilige Fallzeit ist dem \hyperref[ch:Protokoll]{Messprotokoll} zu entnehmen.

\paragraph{Hagen-Poiseuille}
Für diesen Versuchsteil wird nun die Kapillare am unteren Zylinderende betrachtet. Wird das Ventil geöffnet, so ströt das PEG durch die Kapillare in einen Messzylinder. Nachdem sich nach dem Öffnen ein konstanter Massenfluss eingestellt hat, wird eine Zeitstoppung gestartet. Immer wenn sich das Volumen um $5 \mathrm{cm^3}$ verändert hat, wird die aktuelle Zeit aufgeschrieben. Somit lässt sich der Volumenfluss bestimmen.
